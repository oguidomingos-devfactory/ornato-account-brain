# Hanna: Scripts de atendimento + automação WhatsApp — Hanna Ornato

- **Identifier:** DES-11
- **Status:** done
- **Priority:** medium

## Descrição

Criar os scripts de atendimento e fluxos de automação WhatsApp para a Hanna Almeida (Ornato Estudio, tatuadora em Brasília-DF).

**Escopo:**
1. **Mensagem de boas-vindas automática** — resposta instantânea (<1 min) no WhatsApp Business. Texto acolhedor, cardápio de serviços, link para portfólio.
2. **Script de qualificação** — fluxo para a assistente (contratada há ~1 semana) seguir:
   - Identificar estilo desejado
   - Tamanho e local do corpo
   - Orçamento / expectativa de investimento
   - Disponibilidade de agenda
   - Encaminhar para Hanna só leads qualificados
3. **Script de follow-up** — para quem pediu orçamento e não respondeu (24h, 72h, 7 dias).
4. **Script de pós-sessão** — agradecimento, pedido de avaliação Google, indicação.
5. **Template de mensagens-modelo** — para situações comuns (reagendamento, atraso, preparo pré-tattoo, cuidados pós).
6. **Guia de objeções no atendimento** — respostas para "tá caro", "vou pensar", "agenda muito longe".

**Contexto:**
- Hanna demora para responder (sessões longas, não responde meia-noite)
- Assistente nova, precisa de processo estruturado
- Ticket médio ~R$1.000, público premium desejado
- Clientes desqualificados é dor recorre
